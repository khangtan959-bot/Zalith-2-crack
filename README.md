Zalith Launcher 2 - Unlocked Edition This is a modified (fork) version of the original Zalith Launcher 2 project by MovTery. The purpose of this mod: Unlocks the Local Account feature for users in Vietnam and IP-blocked regions. Bypasses the mandatory Microsoft license checks when launching the game in Offline mode. Main changes: Edited AccountsManager.kt to bypass the geographic region check (Greater China/Vietnam). Edited LaunchGame.kt to force Offline launch mode. Edited LaunchArgs.kt to configure standard Java parameters for Local accounts. License: This project complies with the GNU GPL v3.0 license of the original project.
